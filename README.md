## 🔄 同步方法

运行同步脚本，将 Obsidian 中的修改同步到 GitHub：

```bash
./sync.sh
```

同步流程：
1. 从 Obsidian vault 复制最新文件
2. 自动提交变更
3. 推送到 GitHub
